# Requests
Eine simple NextJS-Anwendung mit verschiedenen Routen:

    - /home
    - /users
    - /posts 

Die user-Daten und user-Posts wurden mit der JSONPlaceholder API erstellt. 
Für die users Route wurde eine SSR (Server-Side Rendering) und für die posts Route 
eine CSR (Client-Side Rendering) verwendet.

Alle Seiten haben denselben Header mit Links zu den anderen Seiten.

CSS wurde durch Tailwind erstellt.
